🚗 Previsão de Preços de Automóveis: Uma Jornada de Dados
Bem-vindo ao meu projeto de análise e modelagem de dados para o mercado automotivo! Este repositório contém uma exploração completa — desde a limpeza dos dados até a construção de um modelo preditivo — focada em entender o que realmente define o preço de um veículo.

📌 Sobre o Projeto
Comprar ou vender um carro envolve muitas variáveis: quilometragem, ano, marca, estado de conservação... O objetivo deste projeto foi aplicar técnicas de Ciência de Dados para responder a uma pergunta central: Conseguimos prever o preço justo de um carro com base em suas características técnicas?

Este trabalho faz parte do meu portfólio de análise de dados e demonstra habilidades em limpeza de dados, visualização estatística e Machine Learning utilizando Python.

🛠️ Ferramentas Utilizadas
Para garantir uma análise robusta e visualmente elegante, utilizei o ecossistema principal de Data Science:

Pandas & NumPy: Para manipulação e limpeza da base de dados.

Matplotlib & Seaborn: Para criar gráficos que revelam padrões ocultos (usando temas personalizados para melhor leitura).

Scikit-Learn: Para a construção, treino e validação do modelo de regressão.

🚀 O que você encontrará no Notebook?
O arquivo car_price_proj.ipynb está estruturado de forma lógica e narrada:

Análise Exploratória (EDA): Mergulho nas estatísticas descritivas. Descobri, por exemplo, que a média de preços na base estudada gira em torno de R$ 52.638,00.

Visualização: Gráficos de distribuição de preços e correlações para identificar quais fatores mais "pesam" no bolso.

Modelagem: Implementação de um modelo de regressão para realizar as predições.

Avaliação de Performance:

Uso do R² (Coeficiente de Determinação) para entender o quanto o modelo explica a variação dos preços.

Cálculo do RMSE (Erro Quadrático Médio) para medir a precisão das previsões em termos monetários.

Análise de Resíduos: Um passo crucial para validar se o modelo não é tendencioso (bias).

📊 Resultados em Destaque
O projeto não entrega apenas números, mas insights visuais. O gráfico de resíduos incluído ao final permite verificar a diferença entre o Preço Real e o Preço Predito, garantindo a transparência sobre a confiança do modelo.

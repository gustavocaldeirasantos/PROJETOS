Explorando a Indústria de Queijos: Uma Jornada de Dados 🧀
Bem-vindo ao meu projeto de análise de dados sobre a produção de queijos. Mais do que apenas manipular tabelas, este repositório busca entender as nuances por trás de um dos produtos mais tradicionais da indústria alimentícia, utilizando dados reais para descobrir o que define a oferta de queijos atualmente.

🎯 O Objetivo
O foco aqui foi transformar dados brutos em insights compreensíveis. Eu queria responder a perguntas como:

Quão completos são os nossos registros industriais?

Existe uma correlação real entre o tipo de fabricação e a certificação orgânica?

Como a escolha do leite (vaca, cabra, ovelha) influencia o produto final?

🧠 O que foi feito?
1. Limpeza com Propósito
Nenhum dado é perfeito. Identifiquei que colunas cruciais, como as características sensoriais e o tipo de casca, apresentavam mais de 30% de ausência. Em vez de apenas ignorar, essa análise de missing values serviu para entender as limitações da nossa base de dados.

2. A Ciência por trás do Orgânico
Utilizei técnicas de estatística descritiva e cruzamento de dados para identificar "nichos". Através de mapas de calor, consegui visualizar onde os produtos orgânicos ganham força, cruzando o tipo de manufatura com a origem do leite.

3. Visualização que Conta História
Acredito que um bom gráfico deve ser autoexplicativo. Por isso, configurei o ambiente visual com Seaborn e Matplotlib para garantir que as tendências de mercado fossem visíveis à primeira vista.
